#### 项目说明
>* 3.0任务管理
#### 项目结构
```
|--dist       // 打包代码文件夹
|--html       
  |-- src
    |--api    // 接口定义
    |--components
    |--mshowLib     
      |--common   // 公用组件
      |--lib      // 公用库
        |-- Action  // 前进和后退行为处理
        |-- Animate  // 动画类
        |-- animate.ts  // 动画集合
        |-- Background.ts  // 背景基类
        |-- Border.ts  // 边框类
        |-- ColorBackground.ts  // 颜色背景类
        |-- common.ts
        |-- Emit.ts  
        |-- http.ts  
        |-- ImageBackground.ts  
        |-- interface.ts    // 基础定义
        |-- Layer.ts      // 图层类 
        |-- Page.ts       // 页面类
        |-- Task.ts       // 任务类
      |--ui       // 基础组件文件夹 pc端
      |--wxui     // 基础组件文件夹 微信端
    |--pages
      |--pc     // pc端
      |--wx     // 微信端
    |--store    // vuex store定义
    |--api    // 接口定义
|--server
  |-- release   // 任务打包时需要用到的基础文件
    |-- common
      |--css
        |-- animate.min.css    // animate
        |-- common.css    //   每一个页面通用的css文件
      |--js
        |--vue.min.js     // vue
        |--jquery.min.js   // jquery
        |--page_app.js      // 每一个页面的运行文件
      |--page
        |--common.html      // 每一个页面的基础html文件
  |--src
    |--api  // 接口定义
    |--apiRouter  // 后台接口实现
      |-- resources.ts
      |-- task.ts       // 任务增,删,改,查,发布等接口都在这
    |--config
    |--db     // 数据库索引
    |--httpServer
      |--httpServer.ts  // 挂载api
    |--index.ts // 后台主入口
|--tools      // 发布脚本
```


#### 3.0注意事项
##### 与2.0的主要区别
架构不同:
>* 2.0是js架构, 3.0是ts架构
---
加载方式不同:
>* 2.0是加载整个task.json文件
>* 3.0是将task.json拆分为多个html文件
>* 具体实现可参照server/apiRouter/task.ts中 api.releaseTask接口
